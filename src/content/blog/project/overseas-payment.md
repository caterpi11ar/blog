## 为什么大陆银行卡不行

Stripe 是海外独立开发者的首选支付工具，但它**不支持中国大陆**的注册和收款。这不是技术问题，是 Stripe 的合规策略——大陆的跨境收款监管复杂，Stripe 直接放弃了这个市场。

PayPal 虽然支持大陆，但对独立开发者非常不友好：提现周期长、冻结账号的案例比比皆是，且在海外用户的使用率也在下滑。

主流的出路有两条：**香港银行卡注册 Stripe**，或者用 **Merchant of Record（代理收款）** 服务绕开银行账户的问题。

---

## 路径一：香港银行卡 + Stripe

这是最正规、手续费最低的方案。核心步骤：开香港银行账户 → 注册 Stripe HK → 接入产品收款。

### 第一步：开香港数字银行账户

传统香港银行（汇丰、恒生）开户需要本人到柜台，对大陆用户不现实。更可行的是几家**纯数字银行**，全程线上，用大陆证件即可办理：

**ZA Bank（众安银行）**

- 众安保险旗下，持牌数字银行
- 所需材料：内地护照或港澳通行证（回乡证）+ 自拍验证
- 需要一个香港手机号码用于注册
- 全程 App 操作，支持人民币和港币账户，支持国际汇款

**Mox Bank**

- 渣打银行旗下数字银行
- 开户流程与 ZA Bank 类似，同样需要香港手机号
- 界面更简洁，支持 Visa 实体卡

**香港手机号怎么解决**

这是绕不开的门槛。常见做法：
- 计划去香港时顺便买一张本地 SIM 卡（7-Eleven 有售，约 HK$30 起，有效期内充值可保号）
- 淘宝上有代注册的香港虚拟号服务，但稳定性存疑，不推荐用于重要账号

### 第二步：注册 Stripe

有了香港银行账户，注册 Stripe HK 的流程就很标准了：

1. 前往 [stripe.com/hk](https://stripe.com/hk) 注册
2. 账户类型选**个人（Individual）**，不需要香港公司
3. 填写个人信息：护照号、香港地址（如果没有，可以用虚拟地址服务，约 $50～$100/年）
4. 绑定香港银行账户用于收款
5. 提交身份验证（上传证件照 + 自拍），通常 1～3 个工作日审核通过

验证通过后，就可以生成支付链接、接入 Checkout、或者用 API 直接集成到产品里。提现到香港银行账户通常 2 个工作日到账，再从香港账户转回大陆银行需要额外 1～3 天。

### 费用参考

| 项目 | 费用 |
|------|------|
| ZA Bank / Mox 开户 | 免费 |
| Stripe 交易手续费 | 3.4% + HK$2.35（香港本地卡） |
| Stripe 国际卡附加费 | +1.5% |
| 香港虚拟地址（可选） | $50～$100 / 年 |

## 路径二：Merchant of Record（更省事）

如果觉得开香港银行太麻烦，还有一个更省事的方案：用 **Merchant of Record（MoR）** 服务。

MoR 的逻辑是：**他们代替你作为卖家**。用户付款给 LemonSqueezy 或 Paddle，他们处理好支付、税务（VAT/GST）、退款等一切，再把净额打给你。你只需要提供一个可以收国际电汇的账户——**大陆银行账户通常也可以**。

主流选择：

**LemonSqueezy**
- 对独立开发者最友好，界面简单
- 手续费约 5% + $0.50 / 笔
- 支持订阅、一次性购买、积分包等模式
- 收款支持国际电汇，大陆银行可接

**Paddle**
- 功能更完整，适合稍大规模的产品
- 手续费约 5% + $0.50（根据套餐浮动）
- 自动处理全球 VAT/销售税，合规性更好

MoR 的代价是**手续费比 Stripe 高出约 2%**，且对产品的控制权更少（退款、争议由 MoR 处理）。但对于早期产品，能省掉开香港账户这一步是实实在在的省力。

---

## 怎么选

| | Stripe + 香港银行 | LemonSqueezy / Paddle |
|---|---|---|
| 前置成本 | 需要开香港银行账户 | 几乎无门槛 |
| 手续费 | 较低（~3.4%） | 较高（~5%） |
| 税务处理 | 需要自己处理各地 VAT | 自动处理 |
| 适合阶段 | 月流水 $1000+ 之后 | 早期验证阶段 |

**我的建议是：先用 LemonSqueezy 跑通第一笔付费，验证产品的付费意愿。等月流水稳定之后，再花时间开香港银行账户切换到 Stripe，把这 2% 的手续费省回来。**

两者不互斥，切换成本也不高，早期不必为此纠结太久。

---

## 一点提醒

跨境收款涉及外汇管理，大额收入建议了解相关的申报要求。个人通过正规渠道收取境外劳务报酬是合法的，但超过一定金额（通常 $5 万美元/年）需要向银行申报来源。保留好交易记录和合同，遇到银行询问时能说清楚资金性质即可。
