很明显，我们也要单独存储每个房间的地板数据。

在 store 里加一个 floors 数组：


![image.png](/assets/threejs/1e07849c71d1b6f5a4f453b5ff11d938bce6900c.png)

```javascript
floors: [
    {
        points: [
            { x: 0, z: 0},
            { x: 0, z: 800},
            { x: 800, z: 800},
            { x: 800, z: 0},
            { x: 0, z: 0 }
        ]
    }
]
```
因为地板的形状是不规则的：

![image.png](/assets/threejs/a82eff6b7f65cc135b8b98f68c14141b000772df.png)

所以要存储一系列的点。

y 都是 0，所以存储 x、z 就好。

类型也加一下：


![image.png](/assets/threejs/d7d072f3b2b8b0b7520dc4aaf04698bf9709f3b3.png)

```javascript
interface Floor {
    points: Array<{
        x: number,
        z: number
    }>
}

interface State {
    data: {
        walls: Array<Wall>,
        floors: Array<Floor>
    }
}
```
然后我们把它绘制出来：


![image.png](/assets/threejs/59efd31ceda2296efde332e70b7a2a96ad2a637e.png)

用 Shape 把所有的点依次连起来，然后用 ShapeGeometry 画出形状。

```javascript
const floors = data.floors.map(item => {
    const shape = new THREE.Shape();
    shape.moveTo(item.points[0].x, item.points[0].z);
    for(let i = 1; i < item.points.length; i++) {
        shape.lineTo(item.points[i].x, item.points[i].z);
    }

    const geometry = new THREE.ShapeGeometry(shape);
    const material = new THREE.MeshPhongMaterial({
        color: 'orange'
    });
    const floor = new THREE.Mesh(geometry, material);

    return floor;
});
scene.add(...floors);
```

看下效果：


![image.png](/assets/threejs/0e28af23d96a46ea4e07e33738ab6ec5f887f57b.png)

可以看到，形状画出来了，但是角度不对。

设置旋转角度和背面可见：

![image.png](/assets/threejs/e097e757acdedbefef66366c08cd9643723282e6.png)

```javascript
const material = new THREE.MeshPhongMaterial({
    color: 'orange',
    side: THREE.BackSide
});
const floor = new THREE.Mesh(geometry, material);
floor.rotateX(Math.PI / 2);
```

因为 ShapeGeometry、ExtrudeGeometry 这种画出来的几何体的 uv 值都特别大，所以要设置一个比较小的 repeat 才能正常展示出纹理。

```javascript
const textureLoader = new THREE.TextureLoader();
const floorTexture = textureLoader.load('./floor-texture.png');
floorTexture.colorSpace = THREE.SRGBColorSpace;
floorTexture.wrapS =  THREE.RepeatWrapping;
floorTexture.wrapT =  THREE.RepeatWrapping;
floorTexture.repeat.set(0.002, 0.002);
```

地板画好后，继续来画天花板：


![image.png](/assets/threejs/4301ab45a1a23fe321e0ce7344f026f3ac053a8f.png)

```javascript
ceilings: [
    {
        points: [
            { x: 0, z: 0},
            { x: 0, z: 800},
            { x: 800, z: 800},
            { x: 800, z: 0},
            { x: 0, z: 0 }
        ],
        height: 500
    }
]
```
数据这里多一个 height，也就是天花板高度。

加下类型：


![image.png](/assets/threejs/ad32ba2ca073d812bb0ce51e88373e272dc8a997.png)

```javascript
interface Ceiling {
    points: Array<{
        x: number,
        z: number
    }>,
    height: number
}

interface State {
    data: {
        walls: Array<Wall>,
        floors: Array<Floor>,
        ceilings: Array<Ceiling>
    }
}
```

然后画出来：


![image.png](/assets/threejs/5a5b25866f42798dfc8364656603489c9a6df585.png)

```javascript
const ceilings = data.ceilings.map(item => {
    const shape = new THREE.Shape();
    shape.moveTo(item.points[0].x, item.points[0].z);
    for(let i = 1; i < item.points.length; i++) {
        shape.lineTo(item.points[i].x, item.points[i].z);
    }

    const geometry = new THREE.ShapeGeometry(shape);
    const material = new THREE.MeshPhongMaterial({
        color: '#eee',
        side: THREE.FrontSide
    });
    const ceiling = new THREE.Mesh(geometry, material);
    ceiling.rotateX(Math.PI / 2);
    ceiling.position.y = item.height;
    return ceiling;
});
scene.add(...ceilings);
```
颜色比墙深一点，然后设置下 position.y，其余部分一样。

## 总结

这节我们绘制了地板、天花板。

存储一系列的点，用 Shape 画形状，然后用 ShapeGeometry 画出来。

设置单面可见，这样就可以实现从上往下、从下往上只能看到地板或天花板。

墙、门、窗、地板、天花板都完成了，下节就可以来写两室一厅等户型了。
