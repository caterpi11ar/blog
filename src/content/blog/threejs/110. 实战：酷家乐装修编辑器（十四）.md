这节我们继续来绘制 2D 视图的门窗。

在酷家乐里面，门和床的标志如下：

![image.png](/assets/threejs/c81312fe378ceb28635eda394e286c7a6cd234ba.png)

我们也这样来画。

首先，需要把缺口留出来：

![image.png](/assets/threejs/14a227a3f6a0533a36e6355c5c51e91afff6b4fd.png)

这里和 3D 视图还不一样，我们是从 left 到 left + win.width 来挖洞，直接挖穿：

```javascript
item.windows?.forEach(async win => {
    const path = new THREE.Path();

    const { left } = win.leftBottomPosition;

    path.moveTo(left, 0);
    path.lineTo(left, item.depth);
    path.lineTo(left + win.width, item.depth);
    path.lineTo(left + win.width, 0);
    path.lineTo(left, 0);
    shape.holes.push(path);
})
```
看下效果：


![2025-06-26 20.20.23.gif](/assets/threejs/1519504a599894da74f619c262bcf72d8c8885b5.gif)

再看下另一个户型：


![2025-06-26 20.21.17.gif](/assets/threejs/a3a78867f3fd46076cba8b748ea709632e6049b7.gif)

也没啥问题。

然后我们在洞的位置画一个窗户标志就可以了。


![image.png](/assets/threejs/cd8510dad77b46dc3288a067d34164f93691be70.png)

我们用 PlaneGeometry 来画：

![image.png](/assets/threejs/16a6505a60cbeec0c5a0c3009d2638f1e92711ae.png)

```javascript
item.windows?.forEach(win => {
    const geometry = new THREE.PlaneGeometry(win.width, item.depth);
    const material = new THREE.MeshBasicMaterial({
        color: 'green',
        side: THREE.DoubleSide
    });
    const winLogo = new THREE.Mesh(geometry, material);
    wall.add(winLogo);
})
```

![image.png](/assets/threejs/78aed96963cea432d230af9bbc4c48b2949cda1f.png)

它的中心点在最左边，所以应该移动 left +  win.width / 2，这样刚好把中心点移动洞的中间。

![image.png](/assets/threejs/57498c0d99171e3434a419209fddfca5eabf1fe2.png)

```javascript
const { left } = win.leftBottomPosition;
```
```javascript
winLogo.position.x = left + win.width / 2
winLogo.position.y = 100;
```
看下效果：


![image.png](/assets/threejs/3a147966a282e666b3ba4a59615312f6f705d065.png)

切换到另一个户型看下：

![image.png](/assets/threejs/da8264c72f710fb090346e63a7bc813bbc05baef.png)

我们再用同样的方式把门的洞画出来：


![image.png](/assets/threejs/2c70ca61a052903636343a79794548584322b844.png)

```javascript
item.doors?.forEach(async door => {
    const path = new THREE.Path();

    const { left } = door.leftBottomPosition;

    path.moveTo(left, 0);
    path.lineTo(left, item.depth);
    path.lineTo(left + door.width, item.depth);
    path.lineTo(left + door.width, 0);
    path.lineTo(left, 0);
    shape.holes.push(path);
})
```

![image.png](/assets/threejs/fbb6120ebe6e69129f476977457e58e90404f0c7.png)

然后加上蓝色的矩形：


![image.png](/assets/threejs/e74cdfbc6b7080324e0f7ba68ef39309a0121875.png)

```javascript
item.doors?.forEach(door => {
    const { left } = door.leftBottomPosition;

    const geometry = new THREE.PlaneGeometry(door.width, item.depth);
    const material = new THREE.MeshBasicMaterial({
        color: 'blue',
        side: THREE.DoubleSide
    });
    const doorLogo = new THREE.Mesh(geometry, material);
    doorLogo.position.x = left + door.width / 2
    doorLogo.position.y = 100;
    wall.add(doorLogo);
})
```


![image.png](/assets/threejs/2048aa3721e852767b4d1a7d5df7eea898761fcc.png)


![image.png](/assets/threejs/e7cb924199243a2f621274a4d5761ac39c5cf794.png)

当然，门窗都用矩形不好区分，我们也画一个弧形：

![image.png](/assets/threejs/c81312fe378ceb28635eda394e286c7a6cd234ba.png)


![image.png](/assets/threejs/f5972546f2e862588a65fef15b135318810c38d1.png)

```javascript
const shape = new THREE.Shape();
shape.moveTo(0, 0);
shape.arc(0, 0,door.width, 0, Math.PI / 2);
shape.lineTo(0, 0);

const geometry = new THREE.ShapeGeometry(shape);
```

![image.png](/assets/threejs/d2e5a3fb078abe0f79afe98bd7e78c31a60b4bca.png)

形状没啥问题，但中心点在左下角，所以位置改一下：


![image.png](/assets/threejs/7f78cfb94b92b56946b0f8ea5a42b29882a9c4a5.png)

```javascript
doorLogo.position.x = left;
doorLogo.position.z = -100;
```
不需要再移动门宽度的一半了，并且往上移动一些距离，避免和地板重合，出现闪烁现象。

## 总结

这节我们加上了 2D 视图门窗的绘制。

首先在门窗的位置挖洞，这个洞只需要计算 left 和 win.width 就行，整个挖通。

然后在洞的位置加一个矩形 PlaneGeometry 或者 ShapeGeometry。
