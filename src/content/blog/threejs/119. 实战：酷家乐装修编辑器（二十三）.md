这节我们来做家具列表的功能：

![image.png](/assets/threejs/56cd6ce54350ee78c10e89d039c1a3a2afc3771c.png)

之前的家具都是写死在 json 初始数据里的。

实际上家具应该是动态从左侧拖到场景里，然后编辑下位置、角度。

就像酷家乐这样：


![2025-07-08 01.21.27.gif](/assets/threejs/bc9b94de2853129acd8b7d9b61fd8fc27daf42c9.gif)

我们也来实现下这个功能。

首先写下左侧的家具列表：

![image.png](/assets/threejs/d05fd8ad28a79fd7caeb3d65df49e773ccad76e1.png)

我们准备两个家具：

床、餐桌

餐桌就是之前那个，再找一个床的模型：

https://sketchfab.com/3d-models/bed-52da52f882e242aba646980e9757ef9f#download

![image.png](/assets/threejs/42a00c8a70a7976436b369646015f8744799956b.png)

先加载进来看下：

放到 public 目录下：

![image.png](/assets/threejs/2cde1b2ae4c125ff5ed52182bf1278ab2f690650.png)

3D 场景里加载下看看：


![image.png](/assets/threejs/43976ce213e98c9c2eb5bb6e9e654a905c20a195.png)

```javascript
const gltfLoader = new GLTFLoader();
gltfLoader.load('./bed.glb', (gltf) => {
    scene.add(gltf.scene);
    gltf.scene.scale.setScalar(800);
});
```

![image.png](/assets/threejs/ed07df9c6562d0e5e8e2fd3f4e064fd1ff032c4a.png)

之前的餐桌大小刚刚好，我们没调整，但这个床的模型需要放大 800 倍才刚刚好。

我们在模型里也加上这个初始 scale 的数据：


![image.png](/assets/threejs/b25e2ddf44a723d1892ad1f20a701aab6926de95.png)

```javascript
modelScale?: number;
```
不设置默认就是 1

改下数据：


![image.png](/assets/threejs/1f71e62db4d51c7279a2b67a6ff7c747c63f16ec.png)

```javascript
{
    id: "furniture222",
    modelUrl: "./bed.glb",
    modelScale: 800,
    position: {
        x: -647.4215938726065,
        y: 0,
        z: 7164.314346338078
    },
    rotation: {
        x: 0,
        y: 1.482366445885148,
        z: 0
    }
}
```

在 2D 和 3D 家具渲染的时候，设置下这个 scale：


![image.png](/assets/threejs/e25aa5f3135e82eb4e590ae6c90c210cb560e4e4.png)


![image.png](/assets/threejs/176b4760c629ef1f41eeb83e25af0517a253e13f.png)

```javascript
gltf.scene.scale.setScalar(furniture.modelScale || 1);
```

点击切换户型，应用新数据，看下效果：


![2025-07-20 08.28.11.gif](/assets/threejs/7d5868498ff0b812434a1359aa36c1ee349627f6.gif)

渲染出来了。

只是之前地板高度不大对，调一下：


![image.png](/assets/threejs/23094a9bc60419ad6c7ca8e381890f1c9e7dbe29.png)


![2025-07-20 08.32.02.gif](/assets/threejs/79f72b5077855c6728c11e68187a33890a9e3bae.gif)


![2025-07-20 08.33.42.gif](/assets/threejs/81806bb84e22d441e187486e50e5436d75a744f2.gif)


![2025-07-20 08.54.44.gif](/assets/threejs/326f23c692b5be4f8cda2ea45e14bfd93dbf9ca9.gif)

这样就好了。

现在我们是在 json 里写死的数据，接下来把它改成动态添加的家具。

我们先准备两张家具图片：


![image.png](/assets/threejs/fb3ee3cff0e9e40c8d258739f5f363aec30d4ec7.png)

从这个角度截两张图：


![image.png](/assets/threejs/ec4db5b5c7ffef2c597da03eb96db9338804f8e8.png)


![image.png](/assets/threejs/51ace28bd6a2d8dae8935a344a45ac103b904ad2.png)

然后在百度随便搜一张图片，打开这个智能抠图工具：

![image.png](/assets/threejs/afd1ed9a93ed85077e891b33fa50321cdc424734.png)


![2025-07-20 08.40.02.gif](/assets/threejs/618be08dddaeaf588d9251445ce4e0c7d248c045.gif)

点击上传，换成刚截的两张图，就可以得到背景透明的 png 图片了。

下载下来放到 public 目录：


![98d2f27a-74b0-4d36-8e97-0e85dd855a15.png](/assets/threejs/82dbe7abe4bb87cbd95c08ccb261546ecc1e10b8.png)

![3c510157-e853-407c-abd4-f5bdda3165f9.png](/assets/threejs/9712bd6e88e95430ccf951ebae1ea19144963796.png)


![image.png](/assets/threejs/b4ef050d1de45919ed2a8c6eb3a62f8dbe0d7c13.png)

然后来写家具列表：

![image.png](/assets/threejs/a87c90d1299e4e0b76a79c26b97cc7075c122442.png)

```javascript
<Card
    hoverable
    style={{ width: 200, margin: 20 }}
    cover={<img
        width={200}
        src="./bed.png"
    />}
>
    <Meta title="床" description="" />
</Card>
<Card
    hoverable
    style={{ width: 200, margin: 20 }}
    cover={<img
        width={200}
        src="./table.png"
    />}
>
    <Meta title="餐桌" description="" />
</Card>
```
这里不用预览，所以直接用原生 img 标签。

![image.png](/assets/threejs/e923724389d6f01b543bd61ba6f352a6b12fd9d2.png)

这样，家具列表就完成了。

## 总结

这节我们加载了床的模型，并且写了家具列表。

我们用百度的智能抠图工具做的 png 透明背景的抠图，其余用到透明背景图片的地方也可以这样。

家具因为是外部找的，有的需要设置下 scale 才能用，但设置好了就不允许用户调整 scale 了，因为家具大小是不能变的。
