这节正式开始画廊编辑器的开发。

这节我们把布局写一下。

![image.png](/assets/threejs/900b2d5c23c9b079be6ed08fe781507658fd97fc.image)

可以分为 Header、Menu、Main 这三个组件。

创建项目：

```
npx create-vite gallery-editor
```

![image.png](/assets/threejs/7d06335dfefc19307ffceb275f5484c7fc8aa08f.png)

选择 react、typescript 

进入项目，安装依赖和 three.js：

```
pnpm install
pnpm install --save three
pnpm install --save-dev @types/three
```
安装 ant design 组件库：

```
npm install --save antd
```
改下 src/App.tsx

```javascript
import './App.scss'
import Header from './components/Header';
import Menu from './components/Menu';
import Main from './components/Main';

function App() {
  return <div className='wrap'>
    <Header />
    <div className='editor'>
      <Menu/>
      <Main/>
    </div>
  </div>
}

export default App
```

按照之前分析的结构，拆分 Header、Main、Menu 这三个组件，放在 components 目录下。

安装下 sass

```
pnpm install --save-dev sass
```
改下 src/App.scss

```scss
body {
  margin: 0;
}
```

创建三个组件：

src/components/Header/index.tsx
```javascript
function Header() {
    return <div className="Header">Header</div>
}

export default Header;
```

src/components/Main/index.tsx

```javascript
function Main() {
    return <div className="Main">Main</div>
}

export default Main;
```
src/components/Menu/index.jsx

```javascript
function Menu() {
    return <div className="Menu">Menu</div>
}

export default Menu;
```


然后在 main.tsx 里去掉 index.css 和 StrictMode

![image.png](/assets/threejs/a8dbf9d796e761cbd6b2e300382b7bd8ed3b7a49.png)

跑一下：

```
npm run dev
```

![image.png](/assets/threejs/84aaa7cf099e33acef51b640cc399303c78003bb.png)


![image.png](/assets/threejs/61ef0fe26f968ec56318c0a1dfd7f2625c8d501e.png)

然后来写下布局


![image.png](/assets/threejs/b780c6554345b09039a932d378e1214ec0e43261.png)

```scss
.wrap {
  height: 100vh;
  display: flex;
  flex-direction: column;

  .Header {
    height: 60px;
    border-bottom: 1px solid #000;
  }

  .editor {
    flex: 1;
  }
}
```
竖直方向 flex 布局。

整体是 100vh，上面 60px 下面 flex 1

看下效果：

![image.png](/assets/threejs/afc763e9e3fd5d6d66cbda72e8e4981364225b86.png)

然后写下下面部分的布局：

![image.png](/assets/threejs/900b2d5c23c9b079be6ed08fe781507658fd97fc.image)

和之前的酷家乐编辑器一样，下面的 Main 是占整个屏幕宽度。

左边的 Menu 绝对定位，可以展开收起。



![image.png](/assets/threejs/5fbac0ac214b703a707a6784aa48c9390c1ab46b.png)

```scss
.editor {
    flex: 1;

    position: relative;

    .Menu {
      position: absolute;
      top: 0;
      width: 200px;
      height: calc(100vh - 60px);
      background: pink;
    }
    .Menu {
      left: 0;
      border-right: 1px solid #000;
    }
}
```
中间占据 100% 宽度，左侧是固定宽度绝对定位的面板。


![image.png](/assets/threejs/e0d2166e85eef47664e1d042080de15af1744266.png)

这样，布局就完成了。

然后加一下展开收起的按钮：

写个 div：

![image.png](/assets/threejs/9564e6300cc36c8ee74facbdb66d33a5fb0e2eb8.png)

```javascript
<div className="drawer-bar"></div>
```
写下样式：


![image.png](/assets/threejs/571dc403b192d8438d6254a63c39e21518db1298.png)

```scss
.drawer-bar {
  width: 10px;
  height: 80px;
  background: black;
  position: absolute;
  right: -10px;
  top: 50%;
  transform: translateY(-50%);
  cursor:pointer;
}
```
绝对定位，固定宽高。

看下效果：


![image.png](/assets/threejs/1100dea3674158f564f7d84cb0cca660cdc144b0.png)

点击的时候，改下 left

![image.png](/assets/threejs/86dacf067b3c6d088d224e59c85ef4ba60acbd62.png)

```javascript
import { useState } from "react";

function Menu() {
    const [left, setLeft] = useState(0);

    return <div className="Menu" style={{left: left}}>
        Menu
        <div className="drawer-bar" onClick={() => {
            setLeft(left === 0 ? -200 : 0);
        }}></div>
    </div>
}

export default Menu;
```
用 state 保存 left，点击的时候修改 left 为 0 或 -200


![2025-10-19 20.18.47.gif](/assets/threejs/45699cf94f6d8a5dd3f17d99671a306df8dc2081.gif)

给 left 加个过渡效果：


![image.png](/assets/threejs/bbde0704490e7a51d8c20a633b484b88258fe451.png)

```scss
transition: left 0.5s ease-in-out;
```

缓动的动画，0.5s

试一下：


![2025-10-19 20.20.59.gif](/assets/threejs/7a0500e1ad291c48f7be2f828a921991b13f1cef.gif)

平滑多了。


这样，布局就完成了。
