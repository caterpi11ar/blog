上节用采样器加上了很多植被：

![2025-10-26 20.18.00.gif](/assets/threejs/c0a0f1c3fc3b936e7eae88b828a406ebfec11cd2.gif)

但效果还不够真实，我们这节继续完善。

首先加上雾气：

![image.png](/assets/threejs/780ce440ec8f48772edefbfa5bd577aa7d67aaee.png)

```javascript
scene.background = new THREE.Color(0x87CEEB); // 天空蓝
scene.fog = new THREE.Fog(0x87CEEB, 200, 1500);
```
背景是天空蓝。

然后雾气也是从 200 到 1500 的视野距离逐渐加重雾气，颜色也是天空蓝。


![2025-10-26 20.23.36.gif](/assets/threejs/993dfbf1970c02b83ecb8cdbacae834a3d81e319.gif)

然后加上阴影，这样会更真实：


![image.png](/assets/threejs/3f4fef67e8bd90d839d216a0077d6806470e09b5.png)

```javascript
renderer.shadowMap.enabled = true;
```

![image.png](/assets/threejs/97b0716fe02cbbdda59a40b6814d0a22b05d20ef.png)
```javascript
const light = new THREE.DirectionalLight(0xffffff, 0.5);
light.position.set(1000, 3000, 2000);
light.castShadow = true;

light.shadow.mapSize.width = 2048;
light.shadow.mapSize.height = 2048;
light.shadow.camera.near = 0.5;
light.shadow.camera.far = 5000;
light.shadow.camera.left = -1000;
light.shadow.camera.right = 1000;
light.shadow.camera.top = 1000;
light.shadow.camera.bottom = -1000;

scene.add(light);

const ambientLight = new THREE.AmbientLight(0xffffff, 0.1);
scene.add(ambientLight);
```
调了下平行光位置，开启阴影，设置阴影矩阵。

并且加上了一个比较弱的环境光。

然后开启树和花的阴影投射：


![image.png](/assets/threejs/eaaa080a3589165df6e0d0288640e559652106f6.png)


![image.png](/assets/threejs/eb0266d77ca57779108abdf99abf8520a03dc71d.png)

```javascript
tree.traverse((child) => {
    if (child.isMesh) {
        child.castShadow = true;
    }
});
```
```javascript
flower.traverse((child) => {
    if (child.isMesh) {
        child.castShadow = true;
    }
});
```
然后在山坡上开启接收阴影：


![image.png](/assets/threejs/85ca6b9dcdce2799122ad2e7869a56d74575ec97.png)

```javascript
mountainside.receiveShadow = true;
```

![2025-10-26 20.39.10.gif](/assets/threejs/b546b51ef0f4b18dbd37ba5abb1a7dd196634eb8.gif)

加上阴影后，就感觉不那么突兀了，像是那个位置生长出来的一样。

然后加上相机动画：


![image.png](/assets/threejs/2da771222acd5768e753d81136f7ac348275e210.png)

```javascript
let cameraAngle = 0;
const cameraRadius = 800;
const cameraHeight = 400;
```

```javascript
cameraAngle += 0.005;

camera.position.x = Math.cos(cameraAngle) * cameraRadius;
camera.position.z = Math.sin(cameraAngle) * cameraRadius;
camera.position.y = cameraHeight;
camera.lookAt(0, 0, 0);
```

![2025-10-26 20.44.15.gif](/assets/threejs/6e076047cd680e44f0dbf54c76dfba3538be67f7.gif)

## 总结

这节加上了雾、阴影、相机动画。

其实这个效果和之前林海雪原的差不多，但区别在于这次用的是表面采样器。

用表面采样器实现这种效果明显简单了好多，不用自己去计算位置。

并且这次我们用了 5 层噪音，5 种颜色，实现了更复杂的地形和颜色。
