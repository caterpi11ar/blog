这节把桌面部分画一下。

我们要求能点击、滚动，可以做和网页一样的交互。

这种就只能用 CSS3DRenderer 了。

我们先引入下 CSS3DRenderer：

现在不直接把 WebGL 渲染的 canvas 元素挂载到 dom 了：

![image.png](/assets/threejs/a07f77ce11d474176190fc511db9f2a5275f7f97.png)


而是和 CSS3DRenderer 的 dom 一起：


![image.png](/assets/threejs/4a03ea0b3b96cceb972b492d7c696b605866189a.png)

```javascript
const css3Renderer = new CSS3DRenderer();
css3Renderer.setSize(width, height);

const div = document.createElement('div');
div.style.position = 'relative';
div.appendChild(css3Renderer.domElement);
css3Renderer.domElement.style.position = 'absolute';
css3Renderer.domElement.style.left = '0px';
css3Renderer.domElement.style.top = '0px';
css3Renderer.domElement.style.pointerEvents = 'none';

div.appendChild(renderer.domElement);
document.body.appendChild(div);
```
CSS3DRenderer 的 dom 在上面，所以要用绝对定位。

然后它也一样要每帧渲染：

![image.png](/assets/threejs/ec03d3c02325e40647e5cfd9535b9c15ae437c08.png)

```javascript
css3Renderer.render(scene, camera);
```

那直接把 CSS3DObject 加到这个对象下么？

试一下：


![image.png](/assets/threejs/97bccdd66e0cb0530c2e91b25efeaae0f5d0f98b.png)

```javascript
const ele = document.getElementById('desktop');
const css3dObj = new CSS3DObject(ele);
```
```javascript
obj.add(css3dObj);
```

![2025-07-07 09.11.11.gif](/assets/threejs/9a4914f70986c76c1c1d1426a2ad4faaafd91177.gif)

可以看到，明显太大了。

这是因为外面有个 scale：


![image.png](/assets/threejs/0372d129a899b4bf467db847bbb50cab4e1ac3f1.png)

会把所有子对象一起放大。

![image.png](/assets/threejs/1f3782485e4e57c25d81ef69585e3ccdf60a2cac.png)


```javascript
css3dObj.rotateX(-Math.PI / 2);
css3dObj.scale.set(0.01,0.01,0.01);
```
旋转一下，然后缩小 100 倍。

这样放大后就是比较合适的大小了。


![2025-07-07 09.14.35.gif](/assets/threejs/57e2fce07ebc145f693ac27028e3c46e86d473cf.gif)

然后它的位置还要在 y 轴移动一段距离：


![image.png](/assets/threejs/1e123a50fc4030c0cb66356907242c5c49002624.png)


![image.png](/assets/threejs/b35c61ba6231819ab81f33f0258d3dad026750e7.png)

因为放大 100 倍，移动 2 就是 200。

```javascript
css3dObj.position.y = 2;
```

![2025-07-07 09.21.12.gif](/assets/threejs/7bf75243f0370a5f59efccbb496279e5676f2596.gif)

刚刚好。

## 总结

这节我们加上了 CSS3DRenderer。

我们给桌面上加了一个 CSS3DObject，用 html 来渲染内容。

要注意的是整体放大了几百倍，那内容想要大小合适就需要缩小。

我们找到了桌面的对象，添加了一个局部坐标系 AxesHelper，这样调整位置就方便多了。
