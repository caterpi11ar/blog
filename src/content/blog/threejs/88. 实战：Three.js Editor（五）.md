然后按下 delete 键的时候，如果有选中的物体，就要把它删除。

![image.png](/assets/threejs/ef4984995de58fa758533a642c8ace25157d2851.png)

```javascript
useEffect(() => {
    function handleKeydown(e) {
        if(e.key === 'Backspace') {
            sceneRef.current.remove(selectedObj);
        }
    }
    window.addEventListener('keydown', handleKeydown);
    return () => {
        window.removeEventListener('keydown', handleKeydown);
    }
}, [selectedObj]);
```

试一下：


![2025-04-28 19.42.36.gif](/assets/threejs/9cc724d8e7251a4efbf63e717361121ad77c3201.gif)

可以看到，按 delete 键确实删除了。

但 store 里对应的 mesh 没删除。

我们加一个方法：

![image.png](/assets/threejs/22d1e0f373e457616b78fb1cbea908df60a1e13c.png)

```javascript
removeMesh(name) {
    set(state => {
        return {
            data: {
                ...state.data,
                meshArr: state.data.meshArr.filter(mesh => {
                    return mesh.name !== name
                })
            }
        }
    })
},
```
根据 name 来删除 meshArr 里的 mesh。

删除的时候调用下：


![image.png](/assets/threejs/c063eb012f32a44026b7ec7092204288f52eb404.png)

```javascript
if(selectedObj) {
    sceneRef.current.remove(selectedObj);
    removeMesh(selectedObj.name);
}
```

## 总结

这节实现了删除功能。

在选中物体的时候，把它记录到 store 里。

然后按 delete 键的时候，如果有选中的物体，就删除这个物体，并且从 store 里删除，没有的话不做处理。

这样删除就完成了，下节继续来做编辑。
