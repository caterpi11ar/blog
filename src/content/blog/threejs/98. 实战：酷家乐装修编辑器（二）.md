这节开始进入酷家乐装修编辑器的开发。

首先我们写下布局：

![image.png](/assets/threejs/03be18d1ff70cde5bef87b56cb388222b5b6ad28.png)

可以分为 Header、Menu、Main、Properties 这四个组件。

创建项目：

```
npx create-vite home-decoration-editor
```

![image.png](/assets/threejs/6d7234f4419e481cf6637e4ae2f362beba8cef1d.png)

选择 react、typescript （前面一个编辑器用的 js，大的项目还是用 ts 写）

进入项目，安装依赖和 three.js：

```
npm install
npm install --save three
npm install --save-dev @types/three
```
安装 ant design 组件库：

```
npm install --save antd
```
改下 src/App.tsx

```javascript
import './App.scss'
import Header from './components/Header';
import Menu from './components/Menu';
import Main from './components/Main';
import Properties from './components/Properties';

function App() {
  return <div className='wrap'>
    <Header />
    <div className='editor'>
      <Menu/>
      <Main/>
      <Properties/>
    </div>
  </div>
}

export default App
```

按照之前分析的结构，拆分 Header、Main、Menu、Properties 这四个组件：

![image.png](/assets/threejs/03be18d1ff70cde5bef87b56cb388222b5b6ad28.png)

放在 components 目录下。

```
npm install --save-dev sass
```

```scss
 .editor {
    flex: 1;

    position: relative;

    .Menu,.Properties {
      position: absolute;
      top: 0;
      width: 300px;

      left: 0;

    }
    .Properties {
      right: 0;
      background: orange;
      border-left: 1px solid #000;
    }
}
```

看下效果：

![image.png](/assets/threejs/42bd0a039fc675dca15cbda0296f40555c5752fd.png)

点击的时候，改下 left、right

![image.png](/assets/threejs/9979d179b1a4f5e8327271f6cfb46025bc8ed357.png)

```javascript
import { useState } from "react";

function Properties() {

    const [right, setRight] = useState(0);

    return <div className="Properties" style={{right: right}}>
        Properties
        <div className="drawer-bar" onClick={() => {
            setRight(right === 0 ? -300 : 0);
        }}></div>
    </div>
}

export default Properties;
```
用 state 保存 right，点击的时候修改 right 为 0 或 -300

![2025-05-19 11.29.02.gif](/assets/threejs/f1146c40b67df13142fe58659746152df88ea91e.gif)

给 left、right 加个过渡效果：



![image.png](/assets/threejs/5c3370750a924d03518140d195b3afbf32dedc09.png)

```scss
transition: left 0.5s ease-in-out,right 0.5s ease-in-out;
```

缓动的动画，0.5s

![2025-05-19 11.36.21.gif](/assets/threejs/e0d5ec7b46790bb30cae1e25a79292c2fe7ddac2.gif)

另一个组件也是同样，只不过改 left：

```javascript
import { useState } from "react";

function Menu() {

    const [left, setLeft] = useState(0);

    return <div className="Menu" style={{left: left}}>
        Menu
        <div className="drawer-bar" onClick={() => {
            setLeft(left === 0 ? -300 : 0);
        }}></div>
    </div>
}

export default Menu;
```
